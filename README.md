# Resume Timeline

A chronological archive of my resumes from high school graduation through my current Computer Science studies at the University of South Carolina. Each version reflects where I was at that point in time — the skills I had, the experience I'd gained, and how I was presenting myself professionally.

---

## Versions

| Version | Period | Highlights |
|---------|--------|------------|
| V1 | 2023 | First resume — Home Depot associate, self-employed landscaper, freshly graduated high school |
| V2 | 2024 | Started adding CS projects, AMR Services manager role, Phi Iota Alpha leadership |
| V3 | 2024–2025 | Refined formatting, added relevant coursework, cleaner presentation of technical skills |
| V4 | 2025–Present | Active resume — software engineering internship, full-stack projects (LingoQuest, AMR Booking System), expanded tech stack |

---

## File Structure

Each version has two files:
- `.tex` — LaTeX source code
- `.pdf` — Compiled output

---

## How to Compile

Requires a LaTeX distribution (e.g. [TeX Live](https://www.tug.org/texlive/) or [MiKTeX](https://miktex.org/)).

```bash
pdflatex ResumeV4.tex
```

---

## Purpose

This repo exists to show growth — not just in skills and experience, but in how I communicate my value. V1 was written by a high school graduate with a part-time job. V4 was written by a developer with real projects in production.
